# Dépôt flamx.online


## Présentation
Ce dépôt contient le code source du projet **flamx.online**, une application web basée sur le framework Laravel.

## Compétences principales

- **Objets connectés** : Conception de systèmes IoT, capteurs, microcontrôleurs, Arduino, ESP32, Raspberry Pi, intégration hardware/software.
- **Protocoles IoT** : MQTT, HTTP, WebSocket, LoRa, Zigbee, communication temps réel et sécurisée.
- **Cloud & IoT** : Intégration cloud, bases de données temps réel, dashboards et monitoring.
- **Smart City** : Solutions connectées pour villes intelligentes, automatisation, gestion de données urbaines.
- **PHP & Laravel** : Développement back-end, API, sites dynamiques, sécurité, Eloquent ORM.
- **Livewire** : Composants dynamiques Laravel, interactions réactives sans JavaScript.
- **FilamentPHP** : Admin panels élégants et puissants pour Laravel.
- **Expo** : Développement d’applications mobiles React Native.
- **Tailwind CSS** : Interfaces modernes et responsives.

## Installation Laravel

1. Clonez le dépôt :
	```bash
	git clone https://github.com/bnfidele/website.git
	cd website
	```
2. Installez les dépendances PHP :
	```bash
	composer install
	```
3. Copiez le fichier d'environnement :
	```bash
	cp .env.example .env
	```
4. Générez la clé d'application :
	```bash
	php artisan key:generate
	```
5. Configurez la base de données dans le fichier `.env`.
6. Exécutez les migrations :
	```bash
	php artisan migrate
	```
7. (Optionnel) Installez les dépendances front-end :
	```bash
	npm install && npm run dev
	```

## Lancement du serveur de développement
```bash
php artisan serve
```
